🧠 CSS 기초 개념 한 번에 이해하기!

이번 글에서는 CSS에서 자주 쓰이는 폰트, 버튼, Flexbox, Grid, 미디어 쿼리, 아이디, 클래스 등의 개념을 예제와 함께 알아봅니다.

1. 📌 웹 폰트 사용하기

Google Fonts에서 링크를 복사해 <head>에 넣으면 손쉽게 예쁜 글꼴을 쓸 수 있어요.

<link href="https://fonts.googleapis.com/css2?family=Nanum+Gothic&display=swap" rel="stylesheet">

2. 🔗 링크 꾸미기

메뉴에서 기본 파란 링크 대신, 원하는 색상과 여백을 줄 수 있어요.

.nav-links a {
  color: white;
  margin: 0 10px;
  text-decoration: none;
}

3. 🧷 클래스와 아이디 선택자

클래스(.): 여러 요소에 적용 / 아이디(#): 유일한 요소에 적용

#main-header {
  background-color: green;
}
.title {
  font-size: 2rem;
}
